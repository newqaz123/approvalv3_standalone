    Component implementation:

    2. Computed canExport: (requestStatus === 'FinalApproval' || requestStatus === 'Completed') && allApprovalsComplete

       - Clear error, set loading true
       - Call exportRequestAsPDF(requestId)
       - If !success, set error with result.error
       - If success, convert base64 to blob and trigger download:
         * atob() to decode base64
         * Create Uint8Array from character codes
         * Create Blob with type result.contentType
         * Create object URL, create anchor element, click, cleanup
       - Set loading false in finally
    5. Render:
       - Button with onClick={handleExport}, disabled={disabled || loading}, variant="outline", size="sm"
       - If loading: Loader2 icon (animate-spin) + "Generating PDF..." text
       - If not loading: Download icon + "Export PDF" text
       - If error exists: flex row with error message (red-600) + "Try again" button (link variant)

    Use the complete component pattern from research documentation.
