Transform desktop tables into mobile-friendly card layouts.

request lists into touch-friendly card components. Tables don't work well on
mobile due to horizontal scrolling and cramped cells.

Purpose: Enable users to browse requests on mobile with a card-based layout
that displays essential information at a glance.
Output: Working mobile card views for both request table and dashboard table.
</objective>
