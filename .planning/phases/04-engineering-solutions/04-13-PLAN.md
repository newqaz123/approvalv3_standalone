<objective>
Fix file upload authorization to allow assigned engineering users to upload file attachments when submitting solutions.

Purpose: Currently, file upload authorization checks only allow the original requester (request.requesterId !== userId) to upload files. This blocks engineering users who are assigned to work on requests from attaching solution files. The RequestEngineerAssignment table exists but isn't checked in authorization logic.

Output: Engineering users can upload files to requests they're assigned to, enabling solution submission with file attachments.
</objective>
