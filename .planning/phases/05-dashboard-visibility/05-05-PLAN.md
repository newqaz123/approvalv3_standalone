    Purpose: Wrapper around setInterval that handles cleanup properly

    Interface:
    ```tsx
    interface UseIntervalProps {
      callback: () => void
      delay: number | null  // null pauses the interval
    }

    export function useInterval({ callback, delay }: UseIntervalProps): void
    ```

    Implementation:
    - Use useEffect with setInterval
    - Clear interval on unmount or when delay changes
    - When delay is null, interval is paused
    - Use useCallbackRef or similar to ensure callback is stable

    Pattern:
    ```tsx
    useEffect(() => {
      if (delay === null) return
      const id = setInterval(callback, delay)
      return () => clearInterval(id)
    }, [delay, callback])
    ```

    This hook will be used to trigger periodic data refresh.
