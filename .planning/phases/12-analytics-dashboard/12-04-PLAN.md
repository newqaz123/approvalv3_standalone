   - Calculate:

     * minPerRequest: Minimum time
     * maxPerRequest: Maximum time
     * avgPerApprovalLevel: Calculate average time per approval across all approvals

3. Handle empty case:
   - If no completed requests, return all zeros: avg: 0, median: 0, min: 0, max: 0

4. Return in TimeMetrics structure from types/analytics.ts

Calculations must handle edge cases (no data, single request, even/odd counts for median).
