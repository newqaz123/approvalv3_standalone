  Database Integrity Checks                                                                                                                                                 
                                                                                                                                                                            
  After completing tests, verify database integrity:                                                                                                                        
                                                                                                                                                                            
  -- Check all requests have valid foreign keys                                                                                                                             
  SELECT r.id, r.title, u.name as requester, d.name as department                                                                                                           
  FROM requests r                                                                                                                                                           
  LEFT JOIN users u ON r.requester_id = u.id                                                                                                                                
  LEFT JOIN departments d ON r.department_id = d.id;                                                                                                                        
                                                                                                                                                                            
  -- Check all file attachments have valid request references                                                                                                               
  SELECT fa.id, fa.file_name, r.title                                                                                                                                       
  FROM file_attachments fa                                                                                                                                                  
  LEFT JOIN requests r ON fa.request_id = r.id                                                                                                                              
  WHERE r.id IS NULL;                                                                                                                                                       
  -- Should return 0 rows                                                                                                                                                   
                                                                                                                                                                            
  -- Check activity log completeness                                                                                                                                        
  SELECT r.id, r.title, COUNT(ra.id) as activity_count                                                                                                                      
  FROM requests r                                                                                                                                                           
  LEFT JOIN request_activities ra ON r.id = ra.request_id                                                                                                                   
  GROUP BY r.id, r.title                                                                                                                                                    
  HAVING COUNT(ra.id) = 0;                                                                                                                                                  
  -- Should return 0 rows (all requests should have at least "created" activity)

  Known Limitations & Future Improvements                                                                                                                                   
                                                                                                                                                                            
  From Phase 2 summaries:                                                                                                                                                   
                                                                                                                                                                            
  1. Local file storage - Files not replicated like S3                                                                                                                      
    - Concern: Need backup strategy in production                                                                                                                           
    - Concern: Disk space monitoring required                                                                                                                               
  2. File cleanup - Not implemented on request deletion                                                                                                                     
    - Concern: Orphaned files will accumulate                                                                                                                               
  3. No pagination - Request list shows all requests                                                                                                                        
    - Future: Add pagination for users with many requests                                                                                                                   
  4. Static file access - No auth check for downloads                                                                                                                       
    - Future: Implement download endpoint with auth check
